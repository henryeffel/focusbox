# FocusBox

ADHD 친화형 Time Boxing + Interstitial Journaling Chrome Extension입니다.

FocusBox는 브라우저를 켠 순간 사용자가 “지금 무엇을 하려고 열었는지”를 먼저 정하도록 돕습니다. 작업 중 떠오른 생각은 한 줄로 맡겨두고, 사용자를 다시 현재 작업으로 돌려보내는 데 집중합니다.

## 핵심 원칙

- 하루 전체 계획을 요구하지 않습니다.
- 지금 할 작업 하나와 가장 작은 첫 행동만 정합니다.
- 떠오른 생각은 실패로 보지 않고, 나중에 볼 수 있게 맡겨둡니다.
- 방해 사이트는 Focus Session 중에만 부드럽게 멈춥니다.

## 현재 버전

현재 구현 범위는 `v0.2 MVP`입니다.

- 오늘의 첫 박스
- Focus Session 타이머
- 생각 맡기기 / Quick Capture
- 완료 로그
- 중단 로그
- 오늘 기록
- Focus Session 중 방해 사이트 경고
- Popup 기본 화면
- Chrome Storage API 기반 세션/로그 저장
- Manifest V3 service worker 기반 설치/브라우저 시작 처리

기능과 데이터 구조는 단순하게 유지합니다. 저장 키는 `settings`, `currentSession`, `sessions`, `dailyLogs`를 사용합니다.

## 주요 사용자 흐름

1. Chrome 실행 또는 새 탭에서 FocusBox가 열립니다.
2. `오늘의 첫 박스` 화면에서 지금 할 작업과 첫 행동을 입력합니다.
3. `10분`, `25분`, `50분` 중 하나를 선택하고 `첫 박스 시작하기`를 누릅니다.
4. Focus Session 화면에서 현재 작업, 첫 행동, 남은 시간이 크게 표시됩니다.
5. 작업 중 떠오른 생각은 `생각 맡기기`에 한 줄로 저장하고 돌아옵니다.
6. 작업을 끝내면 `완료`, 흐름을 멈춰야 하면 `중단`을 기록합니다.
7. `오늘 기록`에서 시작, 생각 맡김, 완료, 중단, 방해 사이트 로그를 확인합니다.

## 화면별 용어

### 오늘의 첫 박스

브라우저를 켠 이유를 잃어버리기 전에 첫 작업 하나만 정하는 화면입니다.

화면 문구:

```txt
지금은 하나만 정하면 됩니다.
브라우저를 켠 이유를 잃어버리기 전에, 첫 작업 하나만 정해보세요.
```

주요 버튼:

- `첫 박스 시작하기`
- `오늘은 건너뛰기`

### Focus Session

현재 작업과 남은 시간을 가장 먼저 보여주는 집중 화면입니다. `완료`, `생각 맡기기`, `중단` 동작만 제공합니다.

### 생각 맡기기 / Quick Capture

지금 붙잡지 않아도 되는 생각을 한 줄로 저장하고 현재 작업으로 돌아오는 화면입니다.

### 방해 사이트 경고

Focus Session 중 기본 방해 사이트에 접근하면 표시됩니다.

선택 가능한 동작:

- `작업으로 돌아가기`
- `생각 맡기기`
- `기록하고 계속 이동`

### 오늘 기록

오늘의 FocusBox 로그를 시간순으로 확인합니다. `ritual_skipped` 로그는 사용자에게 `첫 박스 건너뜀`으로 표시합니다.

### Popup

진행 중인 박스와 남은 시간을 빠르게 확인하거나, 오늘 기록을 볼 수 있습니다.

## 실행 방법

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위의 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 누릅니다.
4. `focusbox-extension/` 폴더를 선택합니다.

## 테스트 체크리스트

1. 확장 설치 직후 FocusBox 시작 화면이 열리는지 확인합니다.
2. 새 탭을 열어 `오늘의 첫 박스` 화면이 표시되는지 확인합니다.
3. 작업명과 첫 행동을 입력하고 `첫 박스 시작하기`로 세션을 시작합니다.
4. Focus Session 화면에서 현재 작업, 첫 행동, 타이머가 명확히 보이는지 확인합니다.
5. `생각 맡기기`를 저장한 뒤 현재 작업 화면으로 돌아오는지 확인합니다.
6. `완료` 또는 `중단`을 저장하면 오늘 기록에 로그가 추가되는지 확인합니다.
7. Focus Session 중 `youtube.com` 또는 `reddit.com`에 접속하면 경고 화면이 뜨는지 확인합니다.
8. 경고 화면에서 `작업으로 돌아가기`, `생각 맡기기`, `기록하고 계속 이동`이 각각 동작하는지 확인합니다.
9. Popup에서 진행 중인 박스와 오늘 기록이 같은 데이터로 표시되는지 확인합니다.
